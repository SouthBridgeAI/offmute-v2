offmute-v2 — in-browser demo

Drops a media file, extracts audio with ffmpeg.wasm, then runs the full pipeline in-browser: Gemini (inline) for content/diarization/tone, AssemblyAI for timestamps, edit-distance alignment, consistency, finalize. Requires GEMINI_API_KEY and ASSEMBLYAI_API_KEY below.

⚠️ The pure core is verified; ffmpeg.wasm execution must be confirmed in your browser (it fetches ~30MB of wasm core from a CDN on first run). Best for audio that fits Gemini's inline limit (~20MB) — for longer files, use preprocessInBrowser's per-chunk output with the exported per-stage functions.
